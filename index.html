The Founder Archives is a collection of written Volumes set in the world of RWBY, a show owned by Roosterteeth. Each sequential Volume consists of ten chapters, all of which follow the misadventures of Beacon Academy's newly acquired Team Lance.

This project is a non-profit, non-canon fanfic that is not officially connected with Roosterteeth in any way.
